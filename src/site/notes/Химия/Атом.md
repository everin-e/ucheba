#Химия #ЕГЭ-Х1 #ЕГЭ-Х3
**Атом** - химически неделимая частица. **Атомом** является любой **элемент** [[Химия/ПСХЭ Менделеева\|ПСХЭ]].
**Атомы** соединяются в [[Химия/Молекула\|молекулы]].
{ #3670e9}


---
## Строение атома.
**Атом** состоит из **протонов**, **нейтронов** и **электронов**^[**Элементарные частицы,** на которые в химии атом разделить нельзя, но в ядерной физике можно. Дальше **элементарные частицы** (пока что) не делятся.]:
- **Протоны** и **нейтроны** находятся в **ядре атома**.
- **Электроны** находятся на **энергетических уровнях**.

| Частица       | **Заряд** | **Масса**                             |
| ------------- | --------- | ------------------------------------- |
| протон (p)    | +1        | 1 [[Химия/ПСХЭ Менделеева#^b919da\|а.е.м.]] |
| нейтрон (n)   | 0         | 1 [[Химия/ПСХЭ Менделеева#^b919da\|а.е.м.]] |
| электрон (e-) | -1        | 0                                     |

>Кол-во **протонов** (**p**) и **электронов** (**e-**) в **атоме** равно [[Химия/ПСХЭ Менделеева#^df7984\|порядковому номеру]].
>Кол-во **нейтронов** (**n**) в **атоме** равно [[Химия/ПСХЭ Менделеева#^f535ce\|Ar]] - [[Химия/ПСХЭ Менделеева#^4efec4\|Z]].

### Порядок заполнения электронами орбиталей.
>**Орбитали** заполняются в соответствии с тремя принципами:
- **Принцип наименьшей энергии.** Электроны занимают в первую очередь орбитали ближе к ядру (с наименьшей энергией). 
	- ![Pasted image 20250618140035.png](/img/user/files/attachments/Pasted%20image%2020250618140035.png)
	- 1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s  < 5d≈ 4f < 6p < 7s < 6d ≈ 5f < 7p..
- **Принцип Паули.** На орбитали могут находиться не более 2 электронов. Их [[Химия/Атом#^69c11b\|"спины"]] противоположны.
	- ![Pasted image 20250617125957.png](/img/user/files/attachments/Pasted%20image%2020250617125957.png)
- **Правило Хунда.** Орбитали заполняются электронами так, чтобы их суммарный спин был максимальный.
	- ![Pasted image 20250617130007.png](/img/user/files/attachments/Pasted%20image%2020250617130007.png)

### Основное и возбуждённое состояние атома.
**Возбуждение атома** – переход **электронов** [[Химия/Атом#^15b38a\|внешнего энергетического уровня]] с одного [[Химия/Атом#^10f2e8\|подуровня]] на другой, происходящий при поглощении **атомом** энергии извне.^[Это состояние атома является нестабильным, и **электрон** стремится вернуться на исходный, более низкий [[Химия/Атом#^42ae64\|энергетический уровень]], при этом испуская избыточную энергию. (Это уже изучает физика, а не химия)]: 

**Например:** 
- у **Be** на [[Химия/Атом#^15b38a\|внешнем энерг. уровне]] **2** **электрона**, которые полностью заполняют все [[Химия/Атом#^10f2e8\|орбитали]],
	- ![Pasted image 20250618151641.png](/img/user/files/attachments/Pasted%20image%2020250618151641.png)
- в **возбуждённом состоянии** все [[Химия/Атом#^10f2e8\|орбитали]] на [[Химия/Атом#^15b38a\|внешнем энерг. уровне]] становятся **неспаренными**, из-за чего один **электрон** переходит на следующий [[Химия/Атом#^0d25dd\|подуровень]].
	- ![Pasted image 20250618151742.png](/img/user/files/attachments/Pasted%20image%2020250618151742.png)

>Кол-во **неспаренных электронов** в **возбуждённом состоянии** показывает [[Химия/Атом#Валентность атома\|валентность]] элемента.
## Ионы, катионы, анионы.
Когда **атом** теряет или получает **электрон** у него меняется **заряд** и он становится **ионом**.
- **Ионы** - это и **катионы**, и **анионы**.
{ #126752}

- **Катион** - положительно заряженный **атом**, или же **положительный ион**.
{ #126753}

- **Анион** - отрицательно заряженный **атом**, или же **отрицательный ион**.
{ #126754}


>**Ионы** обозначаются химическими символами с верхним индексом, показывающим знак и величину **заряда иона**^[Например: $$\ce{H+,S^2-}$$].

Противоположные **электрические заряды** притягиваются друг к другу, поэтому **катионы** и **анионы** притягиваются друг к другу и образуют **химические соединения** с **ионной связью** ( #TBD)
## Степень окисления атома в соединении.
**Степень окисления (с.о.)** - это условный **заряд**, который показывает сколько **электронов** **атом** бы притянул или отдал, если бы все [[Химия/Химическая связь\|связи]] в соединении были ионными ( #TBD).

| **Высшая**                       | **Промежуточная**                 | **Низшая**                       |
| -------------------------------- | --------------------------------- | -------------------------------- |
| Номер группы                     | #незнаю                           | 8 - номер группы<br>             |
| $$\ce{SO3}$$ с.о. **S** = **+6** | $$\ce{SO2}$$ с.о. **S** = #незнаю | $$\ce{H2S}$$ c.о. **S** = **+2** |

Исключения:
- [[Химия/ПСХЭ Менделеева#^899cb7\|Ме]] всегда имеют положительную с.о.
- [[Химия/ПСХЭ Менделеева#^befd0e\|неМе]] в соединениях с водородом и металлами всегда имеют отрицательную с.о.
- У **H** в **гидридах** с.о. **-1** (в остальном **+1**).
- У **O** в **оксидах** с.о. **-2**, в **пероксидах** **-1**, в соединениях со **F** **+2**
##### Элементы с постоянными степенями окисления:

| **Степень окисления** | **Элементы**                                                     |
| --------------------- | ---------------------------------------------------------------- |
| **-1**                | **F**                                                            |
| **+1**                | [[Химия/ПСХЭ Менделеева#^899cb7\|Ме]] **Ia** группы                    |
| **+2**                | [[Химия/ПСХЭ Менделеева#^899cb7\|Ме]] **IIa** группы + **Zn** и **Cd** |
| **+3**                | **Al**                                                           |
[[Степень окисления.pdf|+Элементы с постоянными степенями окисления в сложных соединениях.]]

> [!Важно!]
> В нейтральной молекуле все окислительные числа в сумме образуют ноль. В ионе их сумма равна заряду иона. 
## Валентность атома.
**Валентность** - число [[Химия/Химическая связь\|химических связей]], которое [[Химия/Атом\|атом]] образует в соединении.
Для большинства элементов **главных подгрупп** валентность можно определить по **номеру группы**.

| **Высшая**                              | **Промежуточная**                       | **Низшая**                              |
| --------------------------------------- | --------------------------------------- | --------------------------------------- |
| Номер группы                            | От низшей до высшей                     | 8 - номер группы                        |
| $$\ce{SO3}$$ Валентность **S** = **VI** | $$\ce{SO2}$$ Валентность **S** = **IV** | $$\ce{H2S}$$ Валентность **S** = **II** |
> Для некоторых [[Химия/ПСХЭ Менделеева#^d2601c\|d-элементов]] и некоторых **элементов** 2-го **периода** валентности определяются немного иначе.

Большинство **элементов** проявляют в **соединениях** переменную **валентность**. Однако для некоторых характерна постоянная **валентность** в соединениях.
#### Постоянную валентность имеют:

| Элемент                                                                  | Валентность | Исключения                                 |
| ------------------------------------------------------------------------ | ----------- | ------------------------------------------ |
| F                                                                        | I           | -                                          |
| O                                                                        | II          | Пероксиды: $$\ce{H2O2, O2F2, K2O2, BaO2}$$ |
| Al                                                                       | III         | -                                          |
| [[Химия/ПСХЭ Менделеева#^899cb7\|Ме]] Ia группы<br>(*Li, Na, K, Rb, Cs, Fr)*   | I           | H (неМе, но тоже I)                        |
| [[Химия/ПСХЭ Менделеева#^899cb7\|Ме]] IIa группы<br>*(Be, Mg, Ca, Sr, Ba, Ra)* | II          | -                                          |

- **Переменную валентность** проявляют все остальные элементы.

## Химические свойства атома ( #TBD):
- [[Химия/ПСХЭ Менделеева#^899cb7\|Металлические]] свойства - способность атома отдавать **электроны**.

{ #8c9373}

- **Энергия ионизации** - минимальное количество энергии, необходимое для удаления электрона из атома или иона в газообразном состоянии. ( #незнаю)
